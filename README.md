A mobile-first AI-powered language learning app focused on spoken communication. This app blends structured curriculum, real-time AI conversations, and voice-based practice to help users actually speak new languages—not just memorize them.

Built with modern tools like Expo, React Native, Supabase, and Gemini (via OpenRouter), the app delivers an immersive and personalized learning experience across multiple languages.

📱 Cross-platform mobile app built with Expo for iOS and Android

🔐 Passwordless authentication using Supabase magic links

🧭 Personalized onboarding based on level, goals, and interests

🌍 Multi-language support (Mandarin, Japanese, Korean, German)

📚 Structured curriculum with chapters and progressive lessons

🎧 Listening practice with native audio and comprehension checks

🎙️ Speaking practice with voice recording and AI transcription

💬 Real-time AI conversations powered by Gemini via OpenRouter

🎯 Scenario-based roleplay with task-driven interactions

✨ AI-generated custom scenarios for dynamic learning (premium)

📊 Progress tracking for speaking, listening, and lesson completion

🔥 Streaks and milestones to boost consistency and motivation
